# Wine-Quality-Prediction-Project
Machine Learning project for predicting wine quality using Random Forest Classifier.

Project Overview
- This project focuses on predicting the quality of wine using Machine Learning based on its physicochemical properties. A Random Forest Classifier was developed to classify wine quality using 11 chemical features. The project demonstrates how machine learning can support efficient, consistent, and data-driven quality assessment.

Objectives
- Predict wine quality using physicochemical properties.
- Develop a multiclass classification model using the Random Forest algorithm.
- Evaluate the model using standard classification metrics.

Dataset
- Source: Kaggle
- Dataset: Wine Quality Dataset
- Machine Learning Task: Multiclass Classification

Input Features
- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol

Target Variable
- Quality (Wine Quality Score)
